# Binary Adder (Python)

## 📌 Description
This project performs addition of two binary numbers and displays the result in binary format.

## 🚀 Features
- Takes binary input from user
- Converts binary to decimal
- Performs addition
- Converts result back to binary

## 🧠 Concepts Used
- Python
- Number system conversion
- Binary operations
- Input/Output
